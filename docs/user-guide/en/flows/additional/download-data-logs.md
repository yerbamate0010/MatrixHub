# Download Data Logs

Navigation: [Home](../../README.md) · [Basic Flows](../../README.md#basic-use-cases) · [Additional Flows](../../README.md#additional-use-cases) · [Reference](../../README.md#reference-sections)

Use this flow to export stored logs for later review or troubleshooting.

## Goal

- find the right month
- download the desired log file
- keep a copy outside the device

## Before You Start

- make sure you know roughly which date range you need
- keep enough free space on your computer or phone for the exported file

## Step 1: Open the Data Logs page

Open `System -> Data Logs`.

The page shows both the archive browser and summary information:

![Data Logs page](../../screenshots/system/data-logs-page.png)

## Step 2: Find the correct month and file

The archive list is grouped by month:

![Data Logs archive list](../../screenshots/system/data-logs-archive-list.png)

Review the month blocks and the file names until you find the period you need.

## Step 3: Download the selected log

Use the download action next to the file you want to keep.

The browser should save the file locally so you can inspect it later or send it
to support.

## Important

- recent issues are usually in the newest month block
- older files stay grouped under earlier months for easier browsing
- exporting a file does not remove it from device storage
- keep the original file name if you want the date to stay obvious during later
  analysis
