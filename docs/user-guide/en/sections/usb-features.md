# USB Features

Navigation: [Home](../README.md) · [Basic Flows](../README.md#basic-use-cases) · [Additional Flows](../README.md#additional-use-cases) · [Reference](../README.md#reference-sections)

The `USB Features` section groups tools that interact with a connected host
computer over USB. Each module now has its own reference page:

- [Air Mouse](usb-features/airmouse.md)
- [Mouse Jiggler](usb-features/mouse-jiggler.md)
- [Keyboard](usb-features/keyboard.md)
- [Macros](usb-features/macros.md)
- [USB Terminal](usb-features/usb-terminal.md)

Start with the page that matches the feature you want to use on the host
computer.

This part of MatrixHub is especially useful when the device is connected to a
PC, mini computer, kiosk, or lab machine and you want quick host control from
the web UI without reaching for extra peripherals.

Access note: on current builds, `Air Mouse`, `Keyboard`, `Mouse Jiggler`,
`Macros`, and `USB Terminal` require management access.

## Air Mouse (Admin only)

Motion-based cursor control with configurable click actions and sensor tuning.

![Air Mouse page](../screenshots/usb-features/airmouse-page.png)

Continue here:

- [Air Mouse reference](usb-features/airmouse.md)

## Mouse Jiggler (Admin only)

Periodic host movement to prevent the connected computer from going idle.

![Mouse Jiggler page](../screenshots/usb-features/mouse-jiggler-page.png)

Continue here:

- [Mouse Jiggler reference](usb-features/mouse-jiggler.md)

## Keyboard (Admin only)

On-screen text and key input sent through the USB keyboard interface, plus a
local physical-keyboard capture monitor for browser-side key tracking.

![Keyboard page](../screenshots/usb-features/keyboard-page.png)

Continue here:

- [Keyboard reference](usb-features/keyboard.md)

## Macros (Admin only)

Saved host automation scripts, optional boot script execution, and command
reference.

![Macros page](../screenshots/usb-features/macros-page.png)

Continue here:

- [Macros reference](usb-features/macros.md)

## USB Terminal (Admin only)

Interactive command console and quick scripts for host-side actions.

![USB Terminal page](../screenshots/usb-features/usb-terminal-page.png)

Continue here:

- [USB Terminal reference](usb-features/usb-terminal.md)

Use USB features when you want to:

- control a host with motion, clicks, or typed input
- automate simple actions with macros
- access terminal-style tools and quick scripts from the web interface
